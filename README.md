# G-Note AI Landing Page

Landing page for G-Note AI - Deployed on GitHub Pages at **www.gnoteai.com**

## Tech Stack

React 19 • TypeScript • Vite 6 • Tailwind CSS 4 • i18next

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Push to `main` branch - GitHub Actions will automatically build and deploy.
